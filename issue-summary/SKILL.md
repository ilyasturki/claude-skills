Generate a focused prompt explaining my current issue with:
- Problem description (2-3 sentences max)
- Only the critical code snippets directly related to the error (with file names)
- The main error message or unexpected behavior
- Minimal steps to reproduce
- What I've tried (bullet points only)
Focus only on what's broken. Omit setup code, imports, and anything that works correctly.
